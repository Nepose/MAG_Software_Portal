App Navigation

The interface of most apps presented in the Main Menu is a vertical list of files and folders (Home Media, Download Manager, Recording Manager) or streams (IPTV Channels).

To move up and down the list, use standard buttons of the remote:

  1. one item up or down – Up / Down buttons;
  2. one page up or down – and buttons (or PgUp/PgDn on the keyboard);
  3. to the beginning of the current folder – Left button (Left or Home on the keyboard);
  4. to the end of the current folder – Right button (Right or End on the keyboard);

Press to select several items, or or . Press to activate quick list search.

Pop-up windows for creating/adding files and streams, system settings and other customized options offer fields with drop-down selection lists of values/parameters. To move between available options use the Left/Right button. The button opens a drop-down list: use the Up/Down buttons for navigation, or the and buttons for scanning through pages. The system supports the quick search function recognising first entered symbols.
