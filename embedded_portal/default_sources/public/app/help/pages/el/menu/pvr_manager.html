Recording Manager

The Recording Manager offers a list of current, planned and completed recordings (since the last device restart) made via the IPTV Channels application (the PVR function).

When you press the button while viewing an IPTV channel, the recording window pops up containing the following fields: recording start and stop time (you can schedule your recordings in advance), its duration, a channel name, media to save the recording, an end file name. If the user lefts the file name field empty, the default name will include the recording start time and *.ts extension. You can add a new recording directly in the Recording Manager using a similar pop-up window. The recorded file will be saved to the selected media. A folder will be created automatically: Records/Channel name/Recording date. It can be accessed via the Recordings folder in the Home Media section.

!Note: the number of active recordings (recordings in process) is limited to five.

Status of recordings in the list is displayed in front of their names.

– still waiting (the recording is scheduled for later time)

– in process (the line with its current status is also displayed)

– unsaved recordings interrupted due to an error (lack of space; wrong time of recording start and/or stop; too many active recordings)

– saved recordings ready for playback when pressed.

Main control keys:

– quick search and filtrate the download list

– add a recording (the window interface is similar to its prototype in the IPTV Channels section). If the file name field is left empty, recording start time becomes the default file name

– select one or more recordings in the list (for example, when deleting old files)

or – display the recording info: a recorded file name (complete or in process), its status, recording start and stop time, a channel name.

- Stop the current recording (selected recordings) still “in process”

The modal menu (press the or button to open) includes three groups of additional settings:

Actions

  • Open — the media player is launched and playback begins (only for recordings with the Complete status);
  • Add recording
  • Edit – to change recording duration, set up its stop time for earlier or later date (only for recordings with the In Process or Scheduled status)
  • Stop – to stop recoding immediately
  • Select (one or several items)
  • Select all (a complete list of tasks)
  • Deselect
  • Invert (selects unselected rows and deselects selected rows)
  • Delete (you can delete both the recording in the Manager and its file saved to external media)

View (only when recordings with different statuses are present in the list)

  • All
  • Errors
  • Completed
  • In process
  • Scheduled

Sort (active with two or more recordings present)

  • By name
  • By time
  • By status
  • By duration

! Important: To start recording you need at least one media connected.

Note: If during content recording the external media runs out of space, the recording will be aborted and marked as an error in the Recording list. The recorded file can be viewed only via the Home Media application (to be found in the Recordings folder) If a message about lack of free space is displayed before the recording starts, you can connect another media and restart the recording using it.
