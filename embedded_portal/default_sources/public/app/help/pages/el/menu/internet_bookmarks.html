Web Bookmarks

The App contains all web addresses saved by the user as bookmarks from a web browser or directly from the app (URLs added manually).

The Web Bookmarks menu option offers quick access to your favorite web sites, which were not pre-defined by the manufacturer in the Web Services section.

Here you can create folders dividing bookmarks into groups by the subject, edit lists, sort, arrange and open web pages for viewing.

Main control keys:

– quick search and filtrate by bookmark names

– add a bookmark (in a pop-up window Create Bookmark the user adds its name and the URL with a domain name being a must, selects one of existing folders or creates a new one to save the bookmark)

– select one or several bookmarks in the list (for example, when deleting old bookmarks)

or – display bookmark info: a user-defined name, web address, time of adding the page to Bookmarks or time of the last edit (changing the name, moving from one folder to another).

The modal menu (press the or button to open) includes two groups of additional settings:

Actions

  • Open — opens the selected bookmark in a web browser
  • Add a bookmark
  • Add a folder (a new folder is created)
  • Edit – to edit a name of a bookmark/folder, URL or a path to a folder
  • Select (one or several items)
  • Select all (a complete bookmark list)
  • Deselect
  • Invert (selects unselected rows and deselects selected rows)
  • Delete (the section also supports deletion with the help of the or ) button)

Sort (active with two or more bookmarks present)

  • By name
  • By time (if a bookmark was moved, renamed, or its URL was changed, it will be the time of the last edit)
