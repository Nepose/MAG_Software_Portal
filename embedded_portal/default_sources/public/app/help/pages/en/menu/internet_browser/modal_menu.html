The modal menu

Press the or button to open the side modal menu consisting of several sections. The Operations tab covers all actions available for a page:

  • Backward
  • Forward
  • Stop
  • Reload
  • Add bookmark
  • Delete bookmark

In the View tab, you can control the page view scale or activate the Fullscreen mode.

The Tools tab supports quick access to the Download Manager and the Web Bookmarks . You can also use it to control the Virtual mouse mode or .
