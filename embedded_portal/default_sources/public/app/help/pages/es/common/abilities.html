Characteristics

The device with pre-installed software offers users the following possibilities:

  • Viewing HD video content
  • Viewing multicast streams (IPTV channels) from a list
  • Creating an IPTV channel list manually
  • Loading a ready channel list via http or from a USB media
  • Previewing IPTV channels in a window
  • Recording broadcasted content to a USB media
  • Converting image format
  • Playing video and audio data from various containers: MPEG-TS, MPEG-PS, avi, mkv, mov, mp4, wmv, mts, m2t, m2ts, divx, m2p, m2v, tp, flv, tod, mod, vob, ts, asf, ac3, mp3, wav, ape, ogg, oga, aiff, wv, tta, wma, flac, m3u8, 3gp, m4v, tts
  • Viewing JPEG, RAW, PNG, BMP, GIF images
  • Decoding MPEG2, MPEG4P2, h264,VC-1, WMV9 video streams
  • Decoding mpeg2-audio, mp3, AC-3 audio streams
  • Displaying mp3 tags
  • Playing media data from so-called home servers
  • Playing media data from USB flash drives and USB HDDs
  • Connecting NFS/CIFS network folders
  • Surfing the web using a built-in browser
  • Playing video and audio content from the web using a direct link or menu-integrated services
  • Downloading any content from the web to a USB media using a direct link
  • Viewing YouTube videos with the help of a built-in app
  • Viewing Picasa galleries with the help of a built-in app
  • Displaying a weather forecast for next few days
  • Three game apps: Lines, Sudoku, and Memory
  • Connecting USB keyboards and USB mice
  • Controlling volume level and mute audio using a remote
  • Low power consumption
