The section in brief

Web Browser, a separate app for web surfing, allows loading any HTML page, searching with Google, or adding your favorite web site to bookmarks. To get access to bookmarks, use the side modal menu (press the or button to display) or go to the Web Bookmarks section. The address line in the upper part of the screen supports two input modes. You can type in an URL and load a web page you need at once or enter a word or a phrase to search with Google.

Use the Up, Down, Left, and Right buttons to navigate the page. Use the and (PgUp / PgDown) buttons to scan through pages. Two modes of page navigation are supported: the smooth cursor (a virtual mouse) or switching between active fields.
