System settings

The System Settings section covers a number of customized options.

Network – Set up connections to wired and wireless networks, PPPoE.

Video – Define video output mode and image resolution; press to access customized options.

Advanced setting – Choose a language, time zone, UPnP client and video stream buffer size. Press to access additional customized options.

Network state – Display current connection settings including web connections and their actual state.

Portal reload – Reload the Main Menu and interface (for example, when choosing language).

Reset settings – Back to original settings and default parameters.

Software update – Choose an update mode: enter an URL when using web (you load an update file from a web site and install it) or enter an external media path (you choose a USB device with an update file to run).

Servers – Set up the time server, the multiportal mode (ability to set up and use third party portals) and other low-level settings.

Audio – Set up initial audio volume (for each launch) and choose audio mode: mono, stereo, Lt/Rt, etc. Press to access customized options.

Keyboard layout — Select a layout language for the virtual keyboard from a list pre-defined by the manufacturer. Use the buttons to set its place in the list, which defines activation priority when switching between languages.

Device info – Display detailed device info: a model, its serial number, МАС address, current software version, etc.

Restart device – Full restart for the device.

Clear user data – Delete all user settings from the device memory: access to e-mail accounts and online services, location for weather forecasts, channel lists and favorites, saved web bookmarks, keyboard layouts, lists of network resources, etc.

! Please be informed that the customized settings displayed by pressing the button should not be edited without direct necessity. Recommended for qualified users only.
