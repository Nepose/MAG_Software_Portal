Только для приставок на базе MAG-270/275.

В меню «DVB каналы» осуществляется сканирование и просмотр DVB каналов.

Предварительно перед сканированием необходимо выбрать корректные настройки DVB в разделе «Главное меню» > «Настройки» > «DVB»

Доступные настройки:

- Тип сигнала: DVB-C, DVB-T, DVB-T2 ( DVB-T2 доступен только для приставок на базе MAG-275).

- Включить питание антенны ( только для DVB-T/T2 )

DVB-T/T2 > Стандарты эфирного наземного цифрового телевидения. ( Используется телевизионная антенна )

DVB-C > Cтандарт организации DVB Project для передачи цифрового телевидения кабелем.

Сканирование DVB каналов осуществляется двумя способами из раздела «DVB каналы» > пункта «DVB меню» (для входа используется кнопка или :

<Автосканирование> или <Сканировать вручную>


Навигация по меню «DVB каналы» аналогична навигации в «IPTV каналы»

В режиме предпросмотра по кнопке или вызывается EPG канала. (если доступно)
