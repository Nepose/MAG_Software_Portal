Подключение сетевых папок

В разделе Home Media есть возможность подключения сетевых папок NFS/SMB. Для этого нужно открыть модальное меню нажатием или на пульте ДУ, выбрать вкладку «Инструменты», а в ней – «Подключить NFS/SMB». В поле «Адрес сервера» введите адрес сервера (например, 192.168.1.10), в поле «Папка на сервере» – путь к ней (например, media/films/funny). Пункт «Локальная папка» определяет имя папки, которая будет отображаться в Home Media, «Тип подключения» - выбирается SMB или NFS. При необходимости введите логин и пароль. После нажатия кнопки на форме в корневом каталоге Home Media будет создан ярлык на сетевую папку. Чтобы отмонтировать сетевую папку, выберите ее в форме, появившейся по нажатию «Отключить NFS/SMB», и нажмите .
