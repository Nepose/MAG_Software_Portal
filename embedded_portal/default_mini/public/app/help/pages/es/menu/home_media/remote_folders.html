Connecting network folders

In the Home Media menu, you can connect NFS/SMB network folders. Press the or button on the remote to open the modal menu. Select the Tools tab, then select the Connect NFS/SMB option. Enter the server address (like 192.168.1.10) in the Server Address field. Enter the server folder path (like media/films/funny) into the Server Folder field. The Local Folder option defines a folder name displayed in the Home Media menu. Select either SMB or NFS for the Connection Type. Enter your login and password if needed.
