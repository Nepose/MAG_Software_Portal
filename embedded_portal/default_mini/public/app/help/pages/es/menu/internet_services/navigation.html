The section in brief and navigation

The menu offers the most demanded and popular TV-adapted web services: photo and video hostings, social networks, search engines, e-mail, info portals, online booking services, etc.

Most web portals use the same interface and navigation as in their mobile versions. To enter a text, use the virtual keyboard or connect a wired or wireless computer keyboard via the USB /WiFi interface. For some web sites with limited support of standard keyboards for navigation a virtual mouse will be a more comfortable variant. With the help of a virtual mouse you can hover the cursor over any point on the page. Press the or button to activate the Virtual mouse function or use the modal menu ( o /Virtual Mouse).
