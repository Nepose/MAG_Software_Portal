TimeShift

The menu item includes settings of the TimeShift function used for delayed playback when watching IPTV channels.

Use TimeShift – The TimeShift function on/off.

Icon On/Off – Displays the TimeShift icon during recording.

File location – Allows you to specify a USB-connected external media for saving video recordings.

TimeShift Maximum Duration – Defines maximum recording time.

Buffer Mode – Cyclic Buffer (default) or Stop When Filled.

Cyclic Buffer – After the pre-set time was specified in the Maximum Duration field, the system starts deleting previously recorded content and replacing it with new data. Perfect for delayed playback of long videos with an external media device of limited capacity.

Stop When Full – When the TimeShift buffer is full, the recording stops automatically and the system displays the ‘Buffer full’ message.

Action When Exit – To Save Buffer, Not to Save Buffer, Save Query. Depending on selected variant, the system saves the recorded content automatically or does not save it. If the Save Query option was selected, a message “Save the buffer content?” is displayed.

Delay – When the TimeShift function is activated, the current IPTV channel is automatically saved to the buffer in 1, 3, 5, or 10 seconds after turning it on. You can set the delay before recording process starts. If the Pause When Button Pressed option was selected, automatic recording is deactivated. To start recording, the user presses the button.
