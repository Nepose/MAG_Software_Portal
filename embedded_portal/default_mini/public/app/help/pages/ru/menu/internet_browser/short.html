Краткое описание

Интернет-браузер представляет собой отдельное приложение для web-серфинга, с помощью которого вы можете загрузить любую html-страницу, осуществить поиск через Google, а также добавить любимый сайт в Закладки (доступны как из модального бокового меню (открывается по нажатию или ), так и из отдельного раздела Интернет-закладки).

Адресная строка вверху экрана поддерживает два режима ввода: вы можете ввести конкретный url и, соответственно, перейти к просмотру нужной вам веб-страницы, или же ввести искомое слово или фразу, по которым запустится поиск в системе Google.

Перемещение по странице осуществляется с помощью навигационных кнопок «вверх», «вниз», «влево» и «вправо», перелистывание страниц – по нажатию и (PgUp/PgDown). Возможен выбор способа перемещения по странице: плавный курсор (виртуальная мышь) или переходы только по активным полям.
