Applications

Applications catalogue

App menu in main screen opens a catalogue of applications. The catalogue contains online services available for installation and further usage. The catalogue of ready-to-install applications(My applications) consists of four main sections:

Video – Paid and free services that are providing access to TV channels, both Ukrainian and global, access to media content using system «Video on Demand» (VoD) and ОТТ technology. The list of available services depends on region, where the device is used, and is updated regularly.

Communication – Most popular and demanded social networks and blog resources adapted for TV screens: Facebook, Livejournal, vKontakte, Odnoklassniki, Twitter, etc.

News – Info portals like Wikipedia and Popular Mechanics, favorite news channels: Korrespondent.net, Lenta.ru, Gazeta.ru, Liga.net, etc.

Utilities – Search systems, e-mail, online booking services, etc.

Using Applications

Before using the application, please, read the terms of use carefully and accept them. The terms of use are opened during the first device startup. Later, you can access them in the Terms of Use menu item.

In order to use an application, select it in the Application catalogue and install:

  • First, go to one of the sections: Video, Social, News, Useful. Point the cursor on an application and press .
  • A new window opens with a short info about the application.
  • To install the application, press again.
  • To launch the application, go the top item in My Applications list.
  • Another way to launch it is to go to the general Catalogue and press twice.

To delete any application from the My Applications list, find the appropriate option in a lower left part of the screen Remove application and select the application you want to delete.

Please, note that after installing an application with paid content you do not get immediate access to the media. First, check the rates and payment terms of the provider and order the selected paid service. Detailed information on service packages and rates can be found on websites of content providers.

Short information is available by pressing or or buttons on RC on marked application.

In order to navigate through the directories and applications USB keyboard/mouse can be used.

Also STB can be operated using application AuraHD Remote that should be installed on Android device (tablet, smartphone...).
