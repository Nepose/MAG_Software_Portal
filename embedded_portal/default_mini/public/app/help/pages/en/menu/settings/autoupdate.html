Software autoupdate

Here you can control software autoupdate settings for upgrading it to the latest version:

  • always on;
  • update to be confirmed by the user;
  • always off.

With the autoupdate option always off, you do not see any messages about new versions. But you can upgrade the software manually if you wish to (use this menu to upgrade).

Two more variants for beta updates are available:

  • On – a beta version will be installed automatically as soon as it appears.
  • Off – the user installs a beta version ‘manually’ by pressing the button.

To check for available software versions ready for installation and updates for them, press the button on the remote. – save changes to the device memory, EXIT – exit without saving

! Important: Beta software stability cannot be guaranteed as it is a test version.
