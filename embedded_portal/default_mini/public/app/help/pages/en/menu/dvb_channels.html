Only for MAG-270/275 based STB.

«DVB channels» can be used for scanning and watching DVB channels.

Before scan it is necessary to select correct settings for DVB in «Main menu» > «Settings» > «DVB»

Available settings:

- Signal type: DVB-C, DVB-T, DVB-T2 ( DVB-T2 available only for STB based on MAG-275).

- Antenna power supply ( only for DVB-T/T2 )

DVB-T/T2 > European-based consortium standard for the broadcast transmission of digital terrestrial television. ( Select if using TV antenna )

DVB-C > European consortium standard for the broadcast transmission of digital television over cable.

To scan for DVB channels enter «DVB channels» > «DVB menu» (to enter menu use button or and select:

<Auto scan> or <Manual scan>


Navigation in «DVB channels» is similar to «IPTV channels»

In preview mode by pressing or button current EPG of channel will be called. (if available)
