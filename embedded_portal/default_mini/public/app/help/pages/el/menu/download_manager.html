Download Manager

The Download Manager is a list of downloads saved by the user. They can be saved from your web browser, from bookmarks, or from the application itself using a direct download link.

When working with a web browser (including the Bookmarks menu item) you can open a direct link to the displayed content or search and select it manually. In the latter case two options are offered: to view the content or to download it. The View option opens a built-in media player and displays the selected content. The Download option opens the Download Manager and the Create Download window with filled-in boxes. Now you can choose the media (if needed) and click To exit the Download Manager and return to the current page in your web browser, press the button on the remote.

Every download has a status: downloading, saving, interrupted due to an error, suspended. Files downloaded successfully are marked with the marker, files still downloaded are marked with . Downloads with errors are marked with , while suspended downloads are marked with . To playback completed downloads with audio or video files, choose the button.

Main control keys:

– quick search and filtrate the download list

– add a download (enter an URL address, a name of a new file, choose external media to save it). If the file name field is left empty, a name will be given automatically. The file name for http://sample.com/myfile.avi web address will be myfile.avi

– select one or several downloads in the list (for example, when deleting old files)

or – display the download info: a file name, URL address for the downloaded file (completed or in process), media to save it (where it was saved), the download status (in process or completed), file size

The modal menu (press the or button to open) includes three groups of additional settings: Actions

  • Open – the media player is launched and playback begins
  • Restart/Pause – to reactivate a delayed download or repeat an erroneous download; to pause a download
  • Add download – to start downloading using a direct link
  • Select (one or several items)
  • Select All (a complete download list)
  • Delete (you can delete a download and its file saved to external media)

View (only when downloads with different statuses are present in the list)

  • All
  • Completed
  • Paused
  • Errors

Sort (active with two or more downloads present)

  • By status
  • By name
  • By size
  • By priority

! Important: to launch a download you need at least one media connected.
