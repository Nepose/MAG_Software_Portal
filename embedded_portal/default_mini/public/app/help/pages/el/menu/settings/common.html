General settings

Playout settings

Using the Settings/Playout option you can choose a default audio track and subtitles language. Defined settings (if any) will be applied to all files or streams opened in a built-in media player (but only if no other customized settings were specified for such files and streams).

Speed test

The option is used to display the state of a given Internet connection (LAN, WiFi) and the current connection speed.

Settings wizard

Your guide to initial launch of the device and its basic set up. Use the Wizard to quickly set up an Internet connection, choose an optimal screen resolution, define your time zone, set up clock and other parameters. Follow its guidelines carefully.

Note: The Settings Wizard is present not in every model/modification. See the technical documentation for detailed info.

Display

Here you can choose a preferred weather forecast view mode (a metric or English system with Celsius or Fahrenheit scale respectively), set up a screensaver and its launch time (idle time before the screensaver appears).
