Terminology

The remote – The remote control.

Image format – TV image width and height ratio (for example 4:3).

CVBS – A composite video output of the device. Corresponds to a RCA-type yellow video input on a TV panel.

Content – Certain data type for video and/or audio playback using TVs.

Video content – File or data stream for video playback.

Audio content – File or data stream for audio playback.

Service – A service or an app for accessing a service.

App – An item in the Main menu of the device giving access to a certain service.
